# SponsorSync Influencer Marketplace

> **Domain:** Marketing / AdTech

## Overview

Brands struggle to find the right creators quickly, while influencers miss relevant opportunities buried in scattered briefs and email threads. Manual shortlisting, back-and-forth negotiation, and unclear expectations inflate cycle times and costs. This project creates a unified space where sponsors publish requirements once and reach a qualified creator base instantly. Influencers can evaluate fit fast and negotiate in-platform. Without consolidation, acquisition costs rise, campaign timelines slip, quality suffers due to mismatched partnerships, and ROI on creator spend erodes.

## Approach

- Conducted discovery workshops to map sponsor and influencer journeys, clarifying roles, workflows, decision points
- Designed Vue.js single-page app with brief intake, search/filter, proposal, negotiation messaging
- Built Flask REST API with SQLite for MVP speed; structured schemas for briefs, bids, chats, audit trails
- Integrated LLM service to auto-summarise briefs and power Q&A bot so creators can query requirements
- Implemented notifications, rate cards, basic dispute/flagging to keep negotiations transparent
- Performed usability tests and closed-loop fixes; staged rollout with instrumentation for activation, conversion, time-to-deal
